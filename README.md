
## Краткое описание

Приложение "Чат".
В первом спринте написан на js + handlebars.
Для стилей использовался sass.

Во втором спринте переписан на ts.
Добавлены линтеры.

## Установка

- `npm i` — установка пакетов,
- `npm run start` — запуск.
